# Athena Text Analytics Lens Lambdas

Detects PII entities and determines the most confident PII type. Amazon Comprehend is the client we use to perform the detection.

## Prerequisites
- Java 1.8+
- Apache Maven

## Development

### Building the project
```
mvn clean install
```

### Run Tests
```
mvn clean test
```

### Clean
```
mvn clean
```
